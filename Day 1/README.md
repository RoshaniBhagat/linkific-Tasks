# Python for Data Engineering

This repository contains Python programs and practice exercises completed while learning the fundamentals of Data Engineering. The project covers Python basics, file handling, date and time operations, and practical exercises for working with data.

## Topics Covered

### Python Basics

* Variables and Data Types
* Lists, Tuples, Sets, and Dictionaries
* List and Dictionary Comprehensions
* Functions and Lambda Functions
* Exception Handling
* Context Managers

### File Handling

* Text Files
* CSV Files
* JSON Files
* Excel Files
* File Paths with `pathlib`
* Bulk File Processing

### DateTime

* Current Date and Time
* Date Parsing and Formatting
* Time Zone Handling
* Date Calculations
* Time Series Basics

### Practice Exercises

* Combine Multiple CSV Files
* Parse JSON Data
* Process Log Files
* Date Manipulation
* Error Handling

## Technologies Used

* Python 3
* Pandas
* OpenPyXL
* Jupyter Notebook
* Git & GitHub
* Visual Studio Code

## How to Run

1. Create and activate a virtual environment.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run any Python script:

```bash
python Python_Basics/01_variables.py
```

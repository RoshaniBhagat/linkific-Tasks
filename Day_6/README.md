
## Objective

Learn advanced Pandas concepts used in Data Engineering, including DataFrame merging, time series analysis, Excel automation, and performance optimization for large datasets.

---

## Learning Objectives

* Combine multiple DataFrames using Merge, Join, and Concat.
* Perform SQL-like joins in Pandas.
* Work with DateTime data and Time Series.
* Perform weekly and monthly resampling.
* Apply rolling windows, shift, and lag operations.
* Read and write Excel files.
* Automate Excel report generation.
* Optimize Pandas performance using vectorization.
* Process large CSV files efficiently using chunk processing.

# How to Run

Clone the repository:

```bash

```

```bash

```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run any Python file:

```bash
python 01_merge_basics.py
```


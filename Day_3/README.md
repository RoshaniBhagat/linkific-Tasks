# Day 3 – Web Scraping with Python

##  Overview

This project is part of my **Data Engineering with Python** learning journey.

The goal of this project is to learn how to collect data from websites using Python through web scraping techniques. It covers HTML parsing, BeautifulSoup, Selenium, data extraction, data cleaning, and storing scraped data.

---

## Learning Objectives

- Understand HTML structure
- Learn web scraping fundamentals
- Use BeautifulSoup for parsing HTML
- Use CSS Selectors
- Extract text and attributes
- Scrape multiple pages
- Handle HTML tables
- Follow links automatically
- Scrape JavaScript-based websites using Selenium
- Follow ethical web scraping practices
- Clean scraped data
- Store data in CSV and JSON formats

## ▶️ Installation

Clone the repository

```bash
git clone <repository_url>
```

Move into the project directory

```bash
cd Day_3
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

## ⚖️ Ethical Web Scraping Practices

This project follows responsible web scraping principles.

- Respect robots.txt
- Use User-Agent headers
- Avoid excessive requests
- Add delays when necessary
- Respect website Terms of Service
- Scrape only publicly available data
